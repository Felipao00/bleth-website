# BLETH - Website Institucional

![BLETH](public/images/logo.svg)

Website institucional premium da BLETH, empresa de tecnologia especializada em desenvolvimento de software, automações, sistemas e soluções digitais.

## 🚀 Tecnologias

- **Framework:** Next.js 15
- **Linguagem:** TypeScript
- **Estilização:** Tailwind CSS
- **Animações:** Framer Motion
- **Ícones:** Lucide Icons

## 📦 Instalação

```bash
# Clone o repositório
git clone https://github.com/bleth/website.git

# Entre no diretório
cd bleth-website

# Instale as dependências
npm install

# Copie as variáveis de ambiente
cp .env.example .env.local

# Execute em desenvolvimento
npm run dev
